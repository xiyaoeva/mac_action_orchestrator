# Chrome Shortcuts and AppleScript Key Codes (macOS)

Notes:
- "Key codes" below are AppleScript key code numbers (System Events).
- Modifiers are not key codes; use "using {command down, option down, shift down, control down}".
- Chrome shortcuts can vary by locale or extensions.

## Key Codes (System Events)
Modifiers (not key codes):
- command down
- shift down
- option down
- control down

Common keys:
- enter: 36
- return (same as enter): 36

- left arrow: 123
- right arrow: 124

- up arrow: 126

## Chrome Default Shortcuts (macOS)
Tabs and windows:
- New tab: command + T (key 17)
- New window: command + N (key 45)
- New incognito window: command + shift + N (key 45)
- Reopen last closed tab: command + shift + T (key 17)
- Close tab: command + W (key 13)
- Close window: command + shift + W (key 13)
- Next tab: control + tab (key 48)
- Previous tab: control + shift + tab (key 48)
- Go to tab 1-8: command + 1..8 (keys 18..28)
- Go to last tab: command + 9 (key 25)
- Move tab left: command + option + left arrow (key 123)
- Move tab right: command + option + right arrow (key 124)

Page and navigation:
- Reload: command + R (key 15)
- Hard reload: command + shift + R (key 15)
- Stop loading: escape (key 53)
- Back: command + left arrow (key 123) or command + [ (key 33)
- Forward: command + right arrow (key 124) or command + ] (key 30)
- Open file: command + O (key 31)
- Print: command + P (key 35)
- Save page as: command + S (key 1)
- Find: command + F (key 3)
- Find next: command + G (key 5)
- Find previous: command + shift + G (key 5)
- Zoom in: command + + (key 24)
- Zoom out: command + - (key 27)
- Reset zoom: command + 0 (key 29)
- Full screen: control + command + F (key 3)

Address bar and search:
- Focus address bar: command + L (key 37)
- Open location: command + L (key 37) then type URL, enter
- Open search in new tab: command + enter (key 36)

Bookmarks and history:
- Show bookmarks bar: command + shift + B (key 11)
- Bookmark this tab: command + D (key 2)
- Bookmark manager: command + option + B (key 11)
- History: command + Y (key 16)
- Downloads: command + shift + J (key 38)

Developer tools:
- Toggle DevTools: command + option + I (key 34)
- Console: command + option + J (key 38)
- Inspect element: command + shift + C (key 8)
- View source: command + option + U (key 32)

Misc:
- Open new tab in background: command + click
- Open link in new tab: command + click
- Open link in new window: shift + click
- Quit Chrome: command + Q (key 12)
